What can I build with Beaker?

Beaker adds APIs for Files, SQLite, and Peer-to-Peer. Here's a few examples of what you can do:

Self-hosted Blogging

Use your static site generator to produce your blog, then publish from your device.

P2P Social Feeds

Use the Hypercore API to produce social feeds that users can share directly, without a host.

Integrated Editor

Use the Files and Hyperdrive APIs to create a complete IDE. Write the HTML in one tab, and view your site in the other.

Personal Search

Use the SQLite API to index content from the P2P network and provide full-text search.
