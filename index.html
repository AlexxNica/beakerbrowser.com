Peer-to-Peer File Sharing

That happens to be in a Browser.

Download Beaker

Beaker is an experimental Browser. It adds new technologies for Peer-to-Peer applications while staying compatible with the rest of the Web.

{% comment %}

New APIs

Files
P2P
SQLite

Beaker adds APIs for Files, SQLite, and P2P. Use them with client-side Javascript to save permanent data, run relational queries, and publish P2P websites.

See: Using the APIs.

{% endcomment %}

Browse Web & P2P

Browse the Web seemlessly along-side Peer-to-Peer sites and file-archives.

Publish from Home

Create new sites with one click, and share them on the Hyperdrive network. You only share the sites (and bandwidth) you choose.

See: How to create a site.

Share Secretly

Use Beaker and command-line tools to share files with encrypted connections.

See: How to share secretly.
