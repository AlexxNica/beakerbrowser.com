A Peer-to-Peer Platform

That happens to be a Browser

Download Beaker

Beaker is an experimental Browser. It adds new technologies for Peer-to-Peer applications while staying compatible with the rest of the Web.

New APIs

Files
P2P
SQLite

Beaker adds APIs for Files, SQLite, and P2P. Use them with client-side Javascript to save permanent data, run relational queries, and publish P2P websites.

See: Using the APIs.

New Protocols

"Git meets BitTorrent." The Hyperdrive protocol shares signed, versioned files on a peer-to-peer network.

Fork directly from the browser. Host from home, or push to a public peer. It browses just like HTTP, and you only seed the sites you choose to seed.

See: How to share a site.

Still Old-School

Beaker browses the old-school Web too. We figure that's still a good idea.
