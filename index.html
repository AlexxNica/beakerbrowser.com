Mad Science (n.) Anything you do, then say, "Wait was this a good idea?"

Beaker is a highly opinionated and standards-noncompliant browser. It's built on top of Chrome, and adds features we think the Web needs:

Hostless Websites

Beaker shares sites using the Dat P2P Network, a "Git-meets-BitTorrent" technology created with a public funding from the Knight Foundation.

The dat tool shares files directly from your device. There is no server, and anybody can contribute bandwidth by "seeding." (It's opt-in!) Don't want to host from home? Push to a Public Peer.

See: Howto Create a Site.

New APIs

Files
P2P
SQLite

Beaker breaks from Web standards with the APIs you need want. It adds safe access to Files, SQLite, and Dat, so you can build novel apps without a backend.

Hostless Sites are completely sandboxed. They can ask for new files from the P2P network, but they can't send Ajax requests. That means they can't leak your data without permission! Also, all P2P assets are signed, versioned, and integrity-checked.

See: Using the APIs.

Still Old-School

Beaker browses the old-school Web too. We figure that's still a good idea.
