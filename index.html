Make P2P Apps

In an Experimental Web Browser

Download Beaker

Beaker is a browser for Peer-to-Peer Applications. It adds new technologies while staying compatible with the rest of the Web.

New APIs

Files
P2P
SQLite

Beaker adds APIs for Files, SQLite, and P2P. Use them to save permanent data, run relational SQL queries, and publish P2P sites from your client-side Javascript.

Client/Server should not be the only option. Let's drop the server when we don't need it.

See: Using the APIs.

New Protocols

"Git meets BitTorrent." The Hyperdrive protocol shares signed, versioned files on a peer-to-peer network.

Fork directly from the browser. Host from home, or push to a public peer. It browses just like HTTP, and you only seed the sites you choose to seed.

See: How to share a site.

100% Old-School

Beaker browses the old-school Web too. We figure that's still a good idea.
