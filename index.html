Enjoy Hostless

Beaker shares sites using the Dat P2P Network, a "Git-meets-BitTorrent" technology created with a grant from the Knight Foundation.

The dat tool shares files directly from your device. There is no backend, and anybody can contribute bandwidth by "seeding." (It's opt-in!) Don't want to host from home? Push to a Public Peer.

Learn how to Create Your Site.

Enjoy APIs

Files
P2P
SQLite

Beaker breaks from Web standards with the APIs you need want. It adds safe access to Files, SQLite, and Dat, so you can build novel apps without a backend.

Hostless Sites are completely sandboxed. They can ask for new files from the P2P network, but they can't send Ajax requests. That means they can't leak your data without permission! Also, all P2P assets are signed, versioned, and integrity-checked.

Learn how to Use the APIs.
